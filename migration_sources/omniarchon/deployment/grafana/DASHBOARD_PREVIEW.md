## Dashboard Features

### Interactive Elements
- **Hover tooltips**: Detailed metrics on hover
- **Drill-down**: Click to filter by container
- **Time range sync**: All panels use same time range
- **Auto-refresh**: Live updates every 30 seconds
- **Legend filtering**: Click legend to hide/show series

### Responsive Design
- **Desktop**: Full 24-column layout
- **Tablet**: Panels stack vertically
- **Mobile**: Single column, scrollable

### Export Options
- **PDF**: Print-friendly report
- **PNG**: Screenshot of dashboard
- **JSON**: Dashboard configuration
- **CSV**: Raw metrics data (per panel)
